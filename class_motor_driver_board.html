void MotorDriverBoard::setLeftMotorSpeed ( float desiredSpeed)
inline

Sets the desired speed for the left motor.

This function sets the desired speed for the left motor based on the provided value. If the provided speed is within the valid range [0.0, 1.0], it is set directly. If the provided speed is less than 0.0, the speed is set to 0.0. If the provided speed is greater than 1.0, the speed is set to 1.0.

Parameters
desiredSpeedThe desired speed for the left motor, within the range [0.0, 1.0].
