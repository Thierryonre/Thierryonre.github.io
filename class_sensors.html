void Sensors::updateSensorValues ( void )
inline

Updates the sensor values.

This function updates the values of all the sensors connected to the Sensor object. It reads the sensor values for the far left, left, center, right, and far right sensors by internally calling the read method from the Sensor class for each sensor.
